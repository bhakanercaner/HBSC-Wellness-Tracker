# HBSC Wellness Tracker

## PNUV – Additional Assignment

This project was developed as an additional assignment for the PNUV course at the Faculty of Computer Science and Engineering (FINKI).

The application is inspired by the HBSC (Health Behaviour in School-aged Children) study and aims to promote healthy habits and mental wellbeing among young people.

## Features

- Mood Tracker
- Healthy Habits Check
- Daily Wellness Challenge
- Personal Diary
- Responsive Design
- Interactive User Interface

## Technologies

- HTML5
- CSS3
- JavaScript

## Purpose

The goal of this project is to encourage young people to monitor their daily wellbeing, improve healthy habits, and increase awareness of mental health through a simple and interactive web application.
